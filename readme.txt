== Description ==

* A wordpress plugins to add paypal donation button after each post content*
* Um plgin wordpress para adicionar botoes paypal para todas as paginas e post*
* [Exemplo em](http://andrewebmaster.com.br/doacoes)* 
* It has been developed for not profiting organizations . It comes with options to select weather you want to display it on post , pages or both 
  you can also add some text before your paypal donate banner via options*
* Tem sido desenvolvido para org sem fins lucrativos e desenvolvedores. Ele vem com opções para selecionar o tempo que quiser exibi-lo em post, paginas ou ambos.
  voce tambem pode adicionar algum texto antes de seus paypal doar bandeira de cartao via painel de opcoes. 

== Installation ==

1.Upload `plugin` to the `/wp-content/plugins/` directory
  Carregue `plug-in` para o diretório `/wp-content/plugins/`
2.Activate the plugin through the 'Plugins' menu in WordPress
  Ative o plugin através do menu dos Plugins 'no WordPress
3.go to settings -> donate via paypal and add your options.
  Ir para configurações -> doar via paypal e adicionar as suas opções.
